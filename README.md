# The Grand Barber

A premium luxury barbershop website demo designed for modern grooming businesses.

## Features

* Responsive Design
* Luxury Black & Gold Theme
* Service Showcase
* Customer Reviews
* Gallery Section
* Booking Call-To-Action
* Contact Section
* Mobile Friendly
* Smooth Animations

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

This project was created as a portfolio demonstration for local business website design and development.
